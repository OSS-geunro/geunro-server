## 사전설정: ```.env```
이 프로젝트는 .env 파일을 통한 사전 설정이 필요합니다.

```
#Database

DB_HOST= #DB Address
DB_PORT=3306
DB_USER=geunro
DB_PASSWD= #DB Password
```
